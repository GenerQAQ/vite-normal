## 使用

### 开发

只需要执行以下命令就可以在终端中看到Local地址

```bash
pnpm run dev
```

### 构建

构建该应用只需要执行以下命令

```bash
pnpm run build
```

然后你会看到用于发布的 `dist` 文件夹被生成。
